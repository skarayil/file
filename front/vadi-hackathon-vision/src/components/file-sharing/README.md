# File Sharing Components

Components related to file operations and sharing features.

## Files

- **FileDropzone.tsx**: Drag-and-drop zone for file uploads.
- **FileProgressCard.tsx**: Displays upload progress for a file.
- **ShareModal.tsx**: Dialog for generating and managing share links.
- **RenameModal.tsx**: Dialog for renaming files.

## Usage

These components are primarily used in `FileUpload.tsx` and `MyFiles.tsx`.
